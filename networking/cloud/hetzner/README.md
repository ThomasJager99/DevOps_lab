Hetzner is used in this project as a reliable and cost-effective environment for deploying and testing self-hosted services, including VPN infrastructure and containerized applications.

---

## About

Hetzner is a German cloud and hosting provider offering:

- Virtual private servers (VPS)
- Dedicated servers
- Cloud infrastructure
- High bandwidth and stable networking

Official website: https://www.hetzner.com/

---

## Usage in this project

Hetzner is used to:

- Deploy Xray/VLESS infrastructure
- Test networking and routing configurations
- Run Docker-based services
- Experiment with system administration and DevOps workflows

---

## Notes

All configurations in this directory are adapted for learning, testing, and experimentation purposes.

Sensitive data (IP addresses, keys, UUIDs) is removed or replaced with placeholders.
